De Opleiding

Duur van de opleiding

De opleiding duurt 4 jaar maar je kan hem versneld doen in 3 jaar.

Welke keuzedelen worden aangeboden?

– Smart technology: hier leer je slimme toepassingen te maken door middel van een microcontroller (arduino) en sensoren.


– Voorbereiden op zelfstandig ondernemen: hier leer je meer over het opzetten van een eigen onderneming en hoe je als zzp’er na je studie aan de slag kan.


– Voorbereiden op het hbo: ambities om naar het hbo door te stromen? Middels dit keuzedeel werken we aan een goede aansluiting van mbo naar hbo.

Welke modules behandelen we?

DevSkills - Introductie DevSkills - Scrum UI - Fundamentals Ontwerp - User stories WebDev – Fundamentals Database - SQL beheren DevSkills - GIT UI - Responsive Prototyping WebDev - Apps & OOP BPV - Begeleid werken 1 BPV - Begeleid werken 2 Ontwerp - Functioneel & Technisch WebDev - Verdieping Database - SQL structureren Testen & Verbeteren’ Examen - Software Ontwerp Examen - Software realisatie

Stage

Als je een 3-jarige opleiding doet, loop je 2 stages: 1 van 760 uur en 1 van 560 uur

Als je een 4-jarige opleiding doet, loop je 1 van 760 uur of 2x 560 uur of 1x 760 uur
