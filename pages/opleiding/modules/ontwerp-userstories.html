Ontwerp - User Stories

Project User Stories Leerdoelen:

Deze module heeft als doel de student te leren hoe hij/zij effectief eisen en wensen kan verzamelen bij stakeholders, deze volgens een gestandaardiseerde methodiek kan beschrijven, prioriteren, communiceren naar stakeholders, en controleren bij hen om ervoor te zorgen dat ze correct begrepen zijn.

Training User Stories Leerdoelen:

Tijdens deze training zal de student verschillende methodieken leren kennen voor het vaststellen van user requirements. Daarnaast zal de student vertrouwd raken met de voor- en nadelen van deze methodieken, en leren hoe hij/zij één specifieke methodiek kan toepassen voor het vaststellen van user requirements.
